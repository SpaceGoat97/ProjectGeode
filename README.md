# Edgeware + Vue + Tailwind Starter
This is a simple starter for Edgeware projects. This will help you rapidly prototype with polkadotJS and the Edgeware blockchain. The stack:

- Vue
- Tailwind
- PolkadotJS

This starter comes with a "Connect wallet" button that will allow you to trigger the PolkadotJS extension to tell you your available addresses and account names. This will serve as a solid foundation to 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
